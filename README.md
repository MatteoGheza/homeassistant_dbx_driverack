# DBX DriveRack PA2 - Home Assistant Custom Component

This custom integration connects a DBX DriveRack PA2 DSP to Home Assistant.
It allows you to control it directly from Home Assistant through native entities.

## Notes

- AI was used to help convert the original library into this Home Assistant custom component.
- Testing and validation were performed manually.
- Tested only with `DBX DriveRack PA2`.
